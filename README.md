# 🌟 CLI Habit & Pomodoro Tracker

**Project Title:** CLI Habit & Pomodoro Tracker
**Author:** Tristan Jay Abonita

## 📖 Project Description
The **CLI Habit & Pomodoro Tracker** is a mini-machine project designed to demonstrate proficiency in Python programming. It solves a real-world productivity problem by allowing users to track daily or weekly habits and manage their focus time using a terminal-based Pomodoro timer. 

It heavily utilizes Object-Oriented Programming (OOP) principles, file handling JSON structures, advanced Python concepts (Generators, Decorators, Context Managers), and robust UI error handling.

---

## ✨ Features
* **Habit Management:** Add, view, remove, and mark habits as completed.
* **Smart Sorting:** Automatically lists habits ranked by engagement (uses `lambda` sorting).
* **Pomodoro Timer:** Fully functional built-in terminal countdown timer (25 min work / 5 min break) that saves session data.
* **Daily Summary:** Visual text-based progress bar showing daily habit completion rate.
* **Persistent Storage:** Reads and writes to `data/habits.json` automatically so your progress is never lost.

## 🚀 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone <your-github-repo-link>
   cd Abonita_TristanJay_FinalProject
   ```

2. **Setup the Virtual Environment (Highly Recommended):**
   ```bash
   python -m venv .venv
   
   # For Linux/macOS
   source .venv/bin/activate
   
   # For Windows
   .venv\Scripts\activate
   ```

3. **Install Dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Run the Application:**
   ```bash
   python src/main.py
   ```
